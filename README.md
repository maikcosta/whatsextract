# Whatsextract
Controle de entrada e saída de pessoas do grupo WhatsApp

# Grupos possuem:
    - Arquivo
    - Nome
    - Numero
    - Alunos

# Alunos Possuem:
    - Numero
    - Data de Entrada
    - Data de saída
    - Status => (Saiu/Continua)

# Grupo Precisará :
    - Importar os Grupos
    - Exportar os contatos
    - Extrair o Nome e Número
    - Remover duplicados

# Aluno precisa:
    - Extrair alunos
    - Controlar data de entrada 
    - Controlar data de saída
    - Criar contato
    - Exportar contato
    - Remover duplicados

# Operações
    - Entrada do grupo
    - Saída do grupo
